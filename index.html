How to Play on IOS

  • Download "Surge 5" from the App Store.
  • Tap on "Arrow Next" or after "Default.conf".
  • Tap on Default.
  • Paste this text: 
    [General]
[Rule]
FINAL,DIRECT
[Host]
www.growtopia1.com = 209.97.163.87
www.growtopia2.com = 209.97.163.87
  • Or use this link:
    https://ios.gtpshost.com/mancatopia
  • Press Done. (Optional): Tap on "Rename" and rename that configuration to "MANCA" to save it.
  • Click "Setup" and allow everything.

How to Play on Windows

  • Click on the keyboard and search for notepad.
  • Go to "C:\Windows\System32\drivers\etc". Change "Text documents (*.txt)" to "All files".
  • Click "host" and click "Open". Paste this text and press "Ctrl + S".
  • Open growtopia app.
209.97.163.87 growtopia1.com

How to Play on Host-Go

  • Download HostGo and install the APK.
  • Open HostGo and click "Hosts Editor".
  • Copy this link:
    https://gtpshost.com/gtpsgrowtopia.txt
  • Click "Download and Apply".
  • Back and click start.
  • Open growtopia.

How to Use Power Tunnel

  • Download the Power Tunnel app from here.
  • Install the application and open it.
  • In the settings, paste this link:
    https://gtpshost.com/gtpsgrowtopia.txt
  • Click connect, then run the game.

How to Set Up Virtual Host

  • Download and configure Virtual Host from here.
  • Download Host File.
  • Open Virtual Host and click "SELECT HOST FILE".
  • Select the host file you just downloaded.
  • Click on the large white checkbox and turn it green.
  • Open growtopia.
